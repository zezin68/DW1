Tag code

Ela apresenta seu conteúdo estilizado de maneira a indicar que o texto é um pequeno fragmento de código. Por padrão, o conteúdo é exibido utilizando a fonte monoespaçada padrão do user agent's

Exemplo

        
    x = 10
    if 10 % 2 == 0:
        print("PAR")
    else:
        print("IMPAR")

Tag samp

Ela apresenta seu conteúdo estilizado de maneira a indicar que o texto é um pequeno fragmento de saída de um programa. Seu conteúdo normalmente é renderizado usando a fonte monoespaçada padrão do navegador 

        
    x = 10
    if 10 % 2 == 0:
        print("PAR")
    else:
        print("IMPAR")
        
    A saída será par

Tag kbd

A tag kbd faz com que toda a parte de texto incluída dentro dela seja lida pelo computador como um botão do teclado. Tudo dentro dessa tag é escrito em fonte monoespaçada

ctrl + C : realiza uma cópia para a área de transferência

ctrl + shift + p : abre um terminal com todos os atalhos do vs code

ctrl + v : colo tudo que estiver na sua área de transferência

shift + alt + ↓ : faz criar um cursor a mais para baixo

ctrl + shift + End : seleciona todo texto abaixo de onde o cursor está
